Курс

Оказание первой помощи пострадавшим

Дистанционный курс, предназначенный для всех работников организации, обязанных владеть базовыми навыками оказания первой помощи до прибытия медицинских работников.

Курс соответствует требованиям Приказа Минздрава России № 477н (актуально на 2025 год), ТК РФ ст. 214, 228.1 и может использоваться работодателем в составе системы обучения по охране труда.

Структура курса

Программа курса соответствует перечню обязательных знаний по оказанию первой помощи, включённых в нормативные правовые акты РФ. Содержание может корректироваться с учётом специфики организации.

  1. Правовые основы оказания первой помощи. Понятие первой помощи, обязанности работников, нормативные требования, ответственность.
  2. Алгоритм действий при оказании первой помощи. Оценка обстановки, безопасность спасателя, вызов скорой помощи, действия по схеме SAFE/CАВС.
  3. Первая помощь при отсутствии сознания и дыхания. Проверка сознания, дыхания, восстановление проходимости дыхательных путей, сердечно-лёгочная реанимация.
  4. Первая помощь при кровотечениях. Виды кровотечений, наложение повязок, прямое давление, жгут, правила фиксации времени наложения.
  5. Первая помощь при травмах. Переломы, вывихи, ушибы, иммобилизация, фиксация конечностей, транспортировка.
  6. Первая помощь при ожогах, отморожениях, поражении электротоком. Алгоритмы действий, недопустимые методы, вызов специалистов.
  7. Первая помощь при неотложных состояниях. Судороги, инсульт, инфаркт, удушье, аллергические реакции.
  8. Итоговый модуль: закрепление знаний. Обзор ключевых алгоритмов, проверочные вопросы перед тестом.

Материалы курса

На этой странице размещены методические материалы, используемые при обучении по первой помощи. Работодатель может добавлять собственные видеолекции и инструкции.

Видеолекции

  • Видеолекция 1: Алгоритм SAFE и общие правила безопасности.
  • Видеолекция 2: Оценка состояния пострадавшего.
  • Видеолекция 3: Первая помощь при кровотечениях.
  • Видеолекция 4: СЛР и использование АВД.

Ссылки на видео добавляются ответственным.

Документы и памятки

  • Памятка по оказанию помощи при кровотечениях.
  • Презентация по СЛР.
  • Памятка по действиям при ожогах и электротравмах.

Материалы могут быть дополнены локальными документами работодателя.

Итоговый тест по курсу

После изучения материалов работник проходит итоговое тестирование. Результаты используются для оформления документов о прохождении обучения.

Работник должен корректно указать ФИО, подразделение, должность и организацию.

Сдать итоговый тест по первой помощи

Перед переходом к тесту рекомендуется ознакомиться со всеми материалами курса.

Использование результатов обучения

Результаты тестирования могут использоваться работодателем для оформления журналов, протоколов, приказов и иных документов, связанных с обучением и проверкой знаний работников.

Порядок оформления определяется локальными нормативными актами работодателя.
